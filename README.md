# Travel Companion - React App

A **React.js** travel companion web application that helps users discover places, restaurants, and hotels around the world using real-time data.

## Features

- Search and explore travel destinations
- - Discover nearby restaurants, hotels, and attractions
  - - Interactive map integration
    - - Category filtering (restaurants, hotels, attractions)
      - - Responsive design for all devices
       
        - ## Tech Stack
       
        - - **React.js** (Create React App)
          - - **JavaScript (ES6+)**
            - - **CSS3 / Material UI**
              - - **RapidAPI** (Travel Advisor API / Google Maps API)
               
                - ## How to Run
               
                - ```bash
                  git clone https://github.com/anishbasnetAB/Travel-companion-React.git
                  cd Travel-companion-React
                  npm install
                  npm start
                  ```

                  Opens at `http://localhost:3000`
